Create a full Next.js 14 project using npm with TypeScript and Tailwind CSS. The app is called "SIMON AI" — a mental wellness dashboard titled "THE UNBURDEN" with the tagline: "Your feelings die here. Your peace begins here."

== PROJECT SETUP ==
- Framework: Next.js 14 (App Router)
- Package manager: npm
- Language: TypeScript
- Styling: Tailwind CSS
- Charts: recharts (npm install recharts)
- Icons: lucide-react (npm install lucide-react)
- Theme: DARK theme — background #0E0D0F, card background #1A1820, accent purple #7B5EA7, text white

== FILE STRUCTURE ==
/app
  layout.tsx        — root layout with dark bg
  page.tsx          — main dashboard (Overview page)
/components
  Sidebar.tsx       — left navigation
  TopBar.tsx        — header with title + search + bell
  UserCard.tsx      — profile card (left sidebar bottom)
  MoodChart.tsx     — "Invisible Path" line chart
  PressureChart.tsx — "Pressure Exit" line chart
  LetGoChamber.tsx  — text area with Burn & Release button
  WoundPouch.tsx    — list of wounds/issues
  HonestyDrop.tsx   — 4 mood buttons (Happy, Sad, Angry, Neutral)
  MirrorOfNow.tsx   — mood/stress/progress card
  UnsaidAdvisor.tsx — AI chat widget
